research_snpdb
==============

Script to load SNP and locus data from tab-delimited files into MySQL and MongoDB for performance comparisons.

Prerequistites:
<ul>
  <li>Python</li>
  <li>
    MySQL
    <ul><li>mysql.connector</li></ul>
  </li>
  <li>
    MongoDB
    <ul><li>pymongo</li></ul>
  </li>
  <li>Tab-delimited data files</li>
</ul>
